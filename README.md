# Pet Adoption

[Live Link](https://pet-adoption-f4b82.web.app)

## Project Overview

Pet Adoption is an interactive web application designed to help people adopt pets in need. The platform allows users to browse a variety of pets, view detailed information, and adopt their desired pets. It includes authentication features, pet listings, pet details, and more.

## Features

- **User Authentication**: Users can sign up, log in, and manage their profiles.
- **Pet Listings**: Display a variety of pets available for adoption with detailed information.
- **Search and Filters**: Easily search and filter pets by name and category.
- **Adopt Pets**: Users can view detailed information about each pet and proceed with adoption.
- **Responsive Design**: Mobile-friendly layout to ensure a seamless experience across devices.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Backend**: Firebase Authentication, Firebase Firestore (for user and pet data storage)
- **State Management**: React Context API
